# 🔍 Log Analyzer Tool

A Python-based Apache Log Analyzer designed to detect suspicious activities such as brute-force attacks, web attacks, and unusual traffic patterns.

This project simulates a basic SOC (Security Operations Center) monitoring tool for analyzing server logs.

---

## 🚀 Features

- ✅ Apache log parsing
- ✅ Brute-force attack detection
- ✅ Web attack pattern detection (SQL Injection, XSS attempts)
- ✅ Anomaly detection
- ✅ Traffic summary generation
- ✅ JSON alert reporting
- ✅ CSV traffic report export

---

## 🛠 Technologies Used

- Python 3.x
- JSON
- CSV
- Regular Expressions
